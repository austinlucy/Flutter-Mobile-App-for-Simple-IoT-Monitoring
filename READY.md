- Solution: Add Flutter to PATH or use full path

**Issue: "No devices available"**
- Solution: Create emulator or connect physical device

**Issue: "Build fails"**
- Solution: Run `flutter clean && flutter pub get`

**Issue: "App crashes"**
- Solution: Check console logs with `flutter run -v`
